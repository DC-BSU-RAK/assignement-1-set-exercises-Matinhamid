HTML & CSS Quick Reference Guide

HTML Tags

<h1> - <h6>: Headings

Used to define headings of various levels. <h1> is the highest level, and <h6> is the lowest.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>

<p>: Paragraph

Defines a paragraph of text.

<p>This is a paragraph of text.</p>

<ul> and <li>: Unordered List

Creates an unordered (bulleted) list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>: Ordered List

Creates an ordered (numbered) list.

<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

<a>: Anchor (Link)

Creates a hyperlink to another web page or location.

<a href="https://example.com">Visit Example.com</a>

<img>: Image

Embeds an image in the document.

<img src="image.jpg" alt="Description of image">

<div>: Division

Defines a division or section in an HTML document. Often used as a container.

<div class="container">...</div>

<span>: Inline Span

Defines an inline section in a document. Used to group inline elements.

<p>This is <span style="color:red;">red</span> text.</p>

<header>: Header

Defines a header for a document or section.

<header><h1>Title</h1></header>

<main>: Main

Specifies the main content of a document.

<main><p>Main Content</p></main>

CSS Properties

color: Text Color

Sets the color of the text.

color: blue;

background-color: Background Color

Sets the background color of an element.

background-color: lightgray;

font-size: Font Size

Sets the size of the font.

font-size: 16px;

font-weight: Font Weight

Sets the weight (boldness) of the font.

font-weight: bold;

text-align: Text Alignment

Sets the horizontal alignment of the text.

text-align: center;

padding: Padding

Sets the padding space inside an element.

padding: 10px;

margin: Margin

Sets the margin space outside an element.

margin: 20px;

border: Border

Sets the border of an element.

border: 1px solid black;
